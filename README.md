# lost-found-tracker
A Java-based Lost &amp; Found Management System that allows users to report, search, and match lost and found items using MySQL database integration

# Features
- User Login System
- Add Lost/Found Items
- Search Items
- Match Lost & Found Items
- Delete Items (Admin)
- Timestamp Tracking

# Technologies
- Java
- MySQL
- JDBC

# Key Highlight
Implemented item matching logic using SQL joins to automatically connect lost and found entries.
